# GitHub 首页展示文案

## 版本一：简洁版

基于 `Python + OpenCV + Flask + SQLite` 实现的车牌识别与信息管理系统。项目围绕车辆图片预处理、车牌候选区域定位、结果展示与信息查询展开，适合作为计算机视觉应用实践项目展示。

## 版本二：标准版

这是一个面向车辆出入管理场景的计算机视觉应用实践项目，基于 `Python`、`OpenCV`、`Flask` 和 `SQLite` 实现了从图像预处理、车牌区域定位到结果展示与信息管理的基础流程。项目重点不在复杂算法训练，而在于完整展示图像处理、结果整理、页面展示和系统集成能力。

## 版本三：适合仓库简介

基于 OpenCV 的车牌识别与信息管理系统，支持车辆图片处理、车牌候选区域提取、结果展示与历史记录查询。
