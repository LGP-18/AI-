# 基于 OpenCV 的车牌识别与信息管理系统

## 1. 项目简介

本项目面向车辆出入信息管理场景，基于 `Python`、`OpenCV`、`Flask` 和 `SQLite` 实现一个简单的车牌识别与信息管理系统。

项目重点围绕以下流程展开：

- 图片导入
- 图像预处理
- 车牌区域定位
- 结果展示
- 信息存储与查询

本项目定位为大学生计算机视觉应用实践项目，重点展示对图像处理、目标区域提取、Web 页面展示和基础系统集成的理解。

## 2. 项目目标

- 完成车辆图片的基础图像预处理
- 实现车牌候选区域定位
- 展示处理结果和车牌区域截图
- 将识别记录写入数据库
- 通过 Web 页面查看和管理结果

## 3. 技术栈

- Python
- OpenCV
- Flask
- SQLite
- Pandas

## 5. 功能说明

- 上传车辆图片
- 自动进行灰度化、去噪、边缘检测
- 基于轮廓筛选定位车牌候选区域
- 保存处理后的图片和车牌截图
- 在页面中展示原图、处理图、候选区域和识别结果
- 将结果写入 SQLite 数据库并展示历史记录

## 6. 项目流程

```text
图片上传
  -> 图像预处理
  -> 边缘检测
  -> 轮廓筛选
  -> 车牌候选区域提取
  -> 结果保存
  -> SQLite 记录入库
  -> Flask 页面展示与查询
```
