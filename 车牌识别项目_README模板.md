# 基于 OpenCV 的车牌识别与信息管理系统

## 项目简介

本项目面向车辆出入信息管理场景，基于 `Python`、`OpenCV`、`Flask` 和 `SQLite` 实现一个简单的车牌识别与信息管理系统。项目重点围绕图像预处理、车牌区域定位、识别结果展示和信息查询展开，完成了从图像处理到结果管理的基础应用流程。

该项目定位为计算机视觉应用实践项目，目标是通过一个贴近实际场景的小系统，提升对图像处理、目标区域提取、结果展示和基础系统集成的理解。

## 项目目标

- 完成车辆图片的基础图像预处理
- 实现车牌区域定位与结果提取
- 对识别结果进行整理、存储和查询
- 通过 Web 页面完成结果展示与基础信息管理

## 技术栈

- `Python`
- `OpenCV`
- `Pandas`
- `Flask`
- `SQLite`

## 功能说明

- 车辆图片导入
- 图像灰度化、去噪、边缘检测
- 轮廓筛选与车牌区域提取
- 识别结果整理
- 识别记录展示
- 查询与基础信息管理

## 项目流程

```text
图片输入
  -> 图像预处理
  -> 边缘检测
  -> 轮廓筛选
  -> 车牌区域定位
  -> 识别结果整理
  -> SQLite 存储
  -> Flask 页面展示与查询
```

## 核心实现思路

### 1. 图像预处理

使用 `OpenCV` 对输入图片进行灰度化、去噪和边缘检测，减少背景干扰，提升后续车牌区域定位的可操作性。

### 2. 车牌区域定位

基于轮廓筛选等方法，对图片中的候选区域进行筛查，提取可能的车牌区域，为后续识别和展示提供基础。

### 3. 结果整理

将处理结果进行结构化整理，并结合 `Pandas` 完成基础数据清洗和记录管理。

### 4. 页面展示

使用 `Flask` 搭建简单的 Web 页面，用于展示识别结果、查询记录和查看基础信息。

### 5. 数据存储

使用 `SQLite` 对识别结果进行存储，支持后续查询和简单管理。
